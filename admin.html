🗑️ Extra কনটেন্ট রিসেট করুন

Admin Panel থেকে যোগ করা সব প্রম্পট ও গান মুছে ফেলতে চাইলে:

১

GitHub Token তৈরি করুন

github.com/settings/tokens/new → Note: "Amin AI World" → repo scope → Generate → Token কপি করে উপরে বসান।

২

প্রম্পট যোগ করুন

"প্রম্পট যোগ" ট্যাবে যান → ছবি সিলেক্ট করুন → ফর্ম পূরণ করুন → "সাইটে যোগ করুন" চাপুন → ১-২ মিনিটে সাইটে দেখা যাবে।

৩

গান যোগ করুন

"গান যোগ" ট্যাবে যান → YouTube লিংক দিন → শিরোনাম ও ক্যাটাগরি দিন → "গান যোগ করুন" চাপুন। সাইটের বিনোদন সেকশনে দেখা যাবে।

৪

প্রোফাইল ছবি আপডেট করুন

"প্রোফাইল ছবি" ট্যাবে যান → আপনার ছবি সিলেক্ট করুন → "আপডেট" চাপুন। সাইটের প্রোফাইল সেকশনে নতুন ছবি দেখা যাবে।
