🗑️ Extra কনটেন্ট রিসেট

Admin থেকে যোগ করা সব কনটেন্ট মুছতে চাইলে:

১

GitHub Token তৈরি করুন

github.com/settings/tokens/new → "Amin AI World" লিখুন → repo scope → Generate → Token কপি করুন।

২

ব্লগ পোস্ট লিখুন

"ব্লগ পোস্ট" ট্যাবে যান → কভার ছবি দিন → শিরোনাম, ক্যাটাগরি ও লেখা দিন → "প্রকাশ করুন" চাপুন। ১-২ মিনিটে সাইটে দেখা যাবে।

৩

প্রম্পট যোগ করুন

ছবি আপলোড করুন → ফর্ম পূরণ করুন → "যোগ করুন" চাপুন।

৪

গান যোগ করুন

YouTube লিংক দিন → শিরোনাম ও ক্যাটাগরি দিন → "গান যোগ করুন" চাপুন।
