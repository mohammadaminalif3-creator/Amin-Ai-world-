# Amin AI World — আপডেট গাইড

## যা যা নতুন করা হয়েছে

1. **নিরাপদ Admin লগইন** — কোনো ডিফল্ট পাসওয়ার্ড আর কোডে লেখা নেই।
   প্রথমবার `admin.html` খুললে এটি আপনাকে **নিজের ইমেইল ও পাসওয়ার্ড সেট করতে** বলবে।
   এই তথ্য শুধু আপনার ব্রাউজারে (SHA-256 দিয়ে হ্যাশ করা আকারে) থাকে — কোনো ফাইলে বা GitHub রিপোতে কখনো প্লেইন টেক্সটে যায় না।
   ⚠️ শুধু নিজের ডিভাইসে/ব্রাউজারে এই সেটআপ করুন। অন্য ব্রাউজার বা ডিভাইস থেকে লগইন করতে চাইলে সেখানেও একবার নিজের ইমেইল-পাসওয়ার্ড দিয়ে সেটআপ করতে হবে (প্রথমবার)।

2. **সব সেকশন এক জায়গা থেকে পরিচালনা** — Admin Panel থেকে এখন যোগ/মুছা যাবে:
   AI টুলস, প্রম্পট (ছবিসহ, romantic/man/woman/others ক্যাটাগরি), গান, মুভি লিংক, ইনকাম অ্যাপ, ব্লগ পোস্ট, ইসলামিক বাণী, প্রেরণার বাণী।

3. **২০টি জনপ্রিয় AI টুল** আগে থেকেই যোগ করা আছে (ChatGPT, Claude, Gemini, Kling AI, Seedance 2.0, TikTok Symphony Creative Studio, NotebookLM, Midjourney, Suno, ElevenLabs ইত্যাদি) — ভিজিটর ক্লিক করলেই সরাসরি সেই টুলে চলে যাবে। আপনি Admin Panel থেকে আরও যোগ করতে পারবেন।

4. **আপনার দেওয়া গান, মুভি ও ইনকাম লিংক** সব যোগ করা হয়েছে (একটি সদৃশ/ডুপ্লিকেট গান লিংক বাদ দেওয়া হয়েছে, যেহেতু ১০ ও ১১ নম্বর লিংক একই ছিল)।

## কীভাবে সাইটে আপলোড করবেন

এই ৩টি ফাইল আপনার GitHub রিপোর (`Amin-Ai-world-`) মূল ফোল্ডারে **আগের `index.html`, `admin.html`, `data.json` প্রতিস্থাপন করে** আপলোড করুন:
- `index.html`
- `admin.html`
- `data.json`

GitHub রিপোতে গিয়ে "Add file → Upload files" দিয়ে টেনে ছেড়ে দিলেই হবে, অথবা প্রতিটি ফাইল খুলে কনটেন্ট কপি-পেস্ট করে সেভ করতে পারেন।

## Publish (প্রকাশ) কীভাবে কাজ করে

- Admin Panel-এ যা যোগ করবেন তা প্রথমে শুধু আপনার ব্রাউজারে **খসড়া** হিসেবে থাকে — কেউ দেখবে না।
- উপরে "🚀 সবার জন্য প্রকাশ করুন" চাপলে GitHub Token চাইবে। টোকেন দিলে তা `data.json` ফাইলে আপডেট হয়ে GitHub-এ কমিট হয়ে যায় এবং ১-২ মিনিটে সবার জন্য লাইভ সাইটে দেখা যায়।
- এই টোকেন **কোথাও সংরক্ষণ করা হয় না** — প্রতিবার প্রকাশ করার সময় নতুন করে দিতে হবে। এটি ইচ্ছাকৃতভাবে এভাবে রাখা হয়েছে, যাতে টোকেন ভুলবশত কারো হাতে বা সোর্স কোডে চলে না যায়।
- টোকেন তৈরির নিয়ম Admin Panel-এর "📖 গাইড" ট্যাবে দেওয়া আছে। শুধু আপনার এই একটি রিপোজিটরিতে "Contents: Read & Write" পারমিশনসহ fine-grained টোকেন বানাবেন — পুরো অ্যাকাউন্ট অ্যাক্সেসের টোকেন না।

## জরুরি নিরাপত্তা সীমাবদ্ধতা (সৎভাবে জানানো প্রয়োজন)

আপনার সাইটটি GitHub Pages-এ হোস্ট করা একটি **static সাইট** — এখানে কোনো ব্যাকএন্ড সার্ভার বা ডেটাবেজ নেই। তাই দুটো বিষয় মাথায় রাখবেন:

- আপনার রিপোজিটরি সম্ভবত **পাবলিক**, তাই `index.html` ও `admin.html`-এর পুরো কোড যে কেউ GitHub-এ দেখতে পারবে। পাসওয়ার্ড এখন প্লেইন টেক্সটে কোথাও নেই, কিন্তু এই কোড সম্পাদনা করে কেউ যদি নিজের কম্পিউটারে চালায়, সে ভুয়া লগইন স্ক্রিন তৈরি করতে পারবে না (কারণ পাসওয়ার্ড আপনার ব্রাউজারেই থাকে), তবে GitHub Token-এর মতো স্পর্শকাতর জিনিস কখনো কোনো ফাইলে/কোডে বসাবেন না।
- Admin Panel-এ ঢোকার লিংক জানলেই কেউ লগইন স্ক্রিন দেখতে পারবে, কিন্তু আপনার পাসওয়ার্ড ছাড়া ভেতরে ঢুকতে পারবে না। তারপরও, খুব শক্তিশালী ও অনন্য পাসওয়ার্ড ব্যবহার করুন এবং সেটি অন্য কোথাও ব্যবহার করবেন না।
- সত্যিকারের সর্বোচ্চ নিরাপত্তার জন্য (একাধিক অ্যাডমিন, লগ, রেট-লিমিটিং ইত্যাদি) ভবিষ্যতে একটি ছোট ব্যাকএন্ড (Cloudflare Workers / Vercel Functions) যোগ করার কথা ভাবতে পারেন — বর্তমান সেটআপ একজন ব্যক্তিগত ব্যবহারকারীর জন্য যথেষ্ট নিরাপদ ও ব্যবহারিক।

## ছবি আপলোড সম্পর্কে

প্রম্পট বা ব্লগ পোস্টে ছবি আপলোড করলে তা সরাসরি `data.json`-এর ভেতরে (base64 আকারে) সংরক্ষিত হয় — কোনো আলাদা ইমেজ হোস্টিং লাগে না। তবে সাইট দ্রুত রাখতে প্রতিটি ছবি ৮০০KB-এর নিচে রাখার চেষ্টা করুন (Admin Panel এটি মনে করিয়ে দেবে)।

## নতুন যোগ করা হয়েছে: Google AdSense-এর জন্য প্রস্তুতি

আপনি ইনকামের জন্য AdSense চেয়েছিলেন, তাই সাইটটি এখন AdSense-এর জন্য প্রস্তুত করা হয়েছে:

**নতুন যোগ হওয়া ফাইল:**
- `about.html` — আমাদের সম্পর্কে ও যোগাযোগ পেজ
- `privacy.html` — প্রাইভেসি পলিসি (AdSense-এর জন্য বাধ্যতামূলক)
- `terms.html` — ব্যবহারের শর্তাবলী
- `robots.txt` ও `sitemap.xml` — সার্চ ইঞ্জিনে (Google Search) সাইট খুঁজে পাওয়ার জন্য
- `ads.txt` — AdSense অনুমোদনের পর এখানে আপনার Publisher ID বসাতে হবে (ফাইলের ভেতরে নির্দেশনা লেখা আছে)

**Admin Panel-এ নতুন কী যোগ হয়েছে:**
"⚙️ সেটিংস" ট্যাবে এখন "📢 Google AdSense ও যোগাযোগ তথ্য" নামে একটি নতুন অংশ আছে। AdSense অনুমোদনের পর সেখানে আপনার Publisher ID (`ca-pub-...`) ও Ad Slot ID বসিয়ে "সেভ করুন" ও তারপর "প্রকাশ করুন" চাপলেই সাইটে বিজ্ঞাপন এমনিতেই দেখানো শুরু হয়ে যাবে — কোনো কোড এডিট করা লাগবে না। এখন খালি থাকলে কোনো বিজ্ঞাপন দেখাবে না।

### AdSense-এ আবেদনের ধাপ

1. https://www.google.com/adsense/start/ -এ গিয়ে আপনার সাইটের লিংক দিয়ে একটি অ্যাকাউন্ট খুলুন।
2. Google একটি ছোট কোড দেবে যাচাইয়ের জন্য — সাধারণত এটি স্বয়ংক্রিয়ভাবেই কাজ করবে, কারণ Publisher ID বসালে সাইটে ইতিমধ্যে AdSense স্ক্রিপ্ট যুক্ত হয়ে যাবে।
3. Google আপনার সাইট পর্যালোচনা করবে (কয়েকদিন থেকে কয়েক সপ্তাহ লাগতে পারে)।
4. অনুমোদন পেলে Publisher ID ও Ad Slot ID Admin Panel-এ বসিয়ে দিন।

### সততার সাথে জানানো জরুরি — অনুমোদন নিশ্চিত নয়

Google AdSense সাধারণত এসব বিষয় দেখে:
- **পর্যাপ্ত অরিজিনাল কনটেন্ট** — শুধু লিংকের তালিকা যথেষ্ট না। আপনার ব্লগ পোস্ট, প্রম্পট বিবরণ ইত্যাদি নিয়মিত লিখতে থাকুন — এটাই সবচেয়ে গুরুত্বপূর্ণ।
- **সাইটের বয়স ও কার্যকলাপ** — একদম নতুন সাইট প্রায়ই প্রত্যাখ্যাত হয়; কিছুদিন নিয়মিত আপডেট চালিয়ে যাওয়া ভালো।
- **নীতিমালা মেনে চলা** — পাইরেটেড/কপিরাইট লঙ্ঘনকারী কনটেন্ট থাকলে অনুমোদন পাওয়া যায় না (এই কারণেই আগে মুভি লিংক সেকশন সরিয়ে দেওয়া হয়েছে)।
- **প্রাইভেসি পলিসি ও যোগাযোগ তথ্য থাকা** — এখন যোগ করা হয়েছে।

এসব ঠিক থাকলেও Google-এর অনুমোদন **নিশ্চিত করে বলা যায় না** — এটা সম্পূর্ণ Google-এর সিদ্ধান্ত। প্রত্যাখ্যাত হলে কারণ জানিয়ে ইমেইল দেবে, সেটা ঠিক করে আবার আবেদন করা যায়।
